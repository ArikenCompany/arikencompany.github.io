コマンド一覧

一般ユーザー向けコマンド
!rank : VALORANTのランク
!uptime : 配信経過時間
!member : メンバー
!wins : VALORANTのコンペ直近5試合の勝率
!sens : VALORANTの感度
!now : 現在やっていること
!sns : snsのコマンド
!twitch : twitchのURL
!youtube : youtubeのURL
!twitter : twitterのURL
!crosshair : VALORANTのクロスヘア設定
!edpi : VALORANTのEdpi
!commands : コマンド一覧
モデレーター用コマンド
!cmd on : コマンドを有効にする※1
!cmd off : コマンドを無効にする※1
!cmd add {!コマンド名} {内容} : コマンドを追加する
!cmd edit {!コマンド名} {内容} : コマンドの内容を変更する
!cmd remove {!コマンド名} : コマンドを削除する
!cmd cooltime : 現在のクールタイムの確認※1
!cmd setcooltime {秒数} : クールタイムを設定する※1
※1: これらのコマンドはdiscordでは反応しないのでtwitchのチャットルーム限定
